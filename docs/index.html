Basic usage:

The output file will contain all text files from your directory, with clear headers separating each file:

[content of main.rs]

=== File: src/lib.rs ===
[content of lib.rs]

ğŸ”§ Technical Details

codepack employs several optimization techniques:

  • Parallel Processing: Uses rayon for concurrent directory walking
  • Memory Mapping: Efficiently handles large files using mmap
  • Buffered I/O: Optimized buffer sizes for maximum performance
  • Channel-based Communication: Efficient thread communication with crossbeam
  • Fast Synchronization: Uses parking_lot mutexes for better performance

ğŸ” Binary File Detection

codepack automatically detects and skips binary files using a sophisticated heuristic:

  • Checks for null bytes
  • Analyzes the ratio of non-ASCII characters
  • Samples only the beginning of large files for quick detection
